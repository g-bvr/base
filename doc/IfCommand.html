IfCommand

Purpose

Branching statements, depending on a boolean condition

Variants

UsageDescription
IF condition EXECUTE script1Execute a script if condition is met
IF NOT condition EXECUTE script2Execute a script if condition is not met
IF condition EXECUTE script1 ELSE script2Execute one of two scripts depending on condition
IF condition SET variable TO value1 ELSE value2Set variable to one of two values depending on condition
IF condition THEN *Execute a command if condition is met

Arguments

ArgumentValue
conditiona variable that holds the condition value. The 'ELSE' action is taken if the variable does not exist, is empty, or is one out of no, 0, false
script1the script that is to be execture if the condition is evaluated to true
script2the script that is to be execture if the condition is is evaluated to false
value1the value that is used if the condition is evaluated to true
value2the value that is used if the condition is evaluated to false
variablethe variable that shall be set to a conditional value
*the command to be executed if condition is evaluated to true

Plugin

This command is defined in BasePlugin
