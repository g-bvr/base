BasePlugin

Purpose

Collects various elementary commands (file handling, flow control, sub script execution, loops, etc.)

Commands

CommandDescription
AppendCommandAppend a text to a file
AssembleCommandAppend the contant of a file to another file
CatchCommandCatch exceptions when executing a command, stacktrace and message will be written to files 'stacktrace.txt' and 'exception.txt' in current working directory
CheckFileExistsCommandCheck if file exists and write result into a variable
CleanupCommandCleanup a folder (remove all files and sub-folders, the folder itself is kept))
CopyCommandCopy a file or directory to another place. It is not allowed that source is a file and target is an (already existing) directory. If target is an existing directory and the source is a file, this file will be copied into a new file in tha tsrget directory with same filename. If source is a directory and target is an already existing drectory, the contents of source are copied to the target. If the target directory does not exist, yet, it will be created (including ancestors if needed).
DebugCommandPrint information for the purpose of debugging
EvaluateCommandEvaluate some numeric or string expression
ExecuteCommandExecute a script with a cloned set of variables (variable changes are not visible after return)
ExecuteSharingVariablesCommandExecute a script sharing the variables (variable changes become visible after return)
ForCommandExecute a command for multiple items
GitPullOrCloneCommandPull a changes of a git repository in the current workspace, if it was not cloned yet, clone it from specified remote location
GitSimulateCloneCommandspecifies a folder in current workspace from which git operations are simulated
HttpRequestCommandIssue a http request
IfCommandBranching statements, depending on a boolean condition
DokumentationCommandCreate a set of html file for documentation.
ReturnCommandReturn a value from an executed script back to the calling script. The returned value is accesible in the calling script via the variable RETURN_VALUE
ReadFileCommandRead content of a file into a variable
ReadOptionalMapCommandRead a key value map from a file into variables
ReadVariablesCommandRead multiple variable from a file
SetSubstitutingCommandSet a variable to a value after applying a regex substitution
SetLineInFileMatchingCommandSet a variable to a value according to a line matching a regex with a group
SetRemovingCommandSet a variable to a value after applying a regex substitution
SetCommandSet a variable to a value
SleepCommandSleep (delay script execution)
StepStartCommandMarks the start of a step block (for structured logging)
StepEndCommandMarks the end of a step block (for structured logging)
StepStateCommandSet the state of the current step
