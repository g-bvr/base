ForCommand

Purpose

Execute a command for multiple items

Variants

UsageDescription
FOR FILE filevar IN folder DO scriptExecute the command for all files in specified folder
FOR SUBFOLDER foldervar IN folder DO scriptExecute the command for all subfolders of specified folder
FOR LINE linevar IN file DO scriptExecute the command for all lines of specified text file
FOR FILE filevar IN folder MATCHING pattern DO scriptExecute the command for all files in specified folder matching specified pattern
FOR SUBFOLDER foldervar IN folder MATCHING pattern DO scriptExecute the command for all subfolders of specified folder matching specified pattern
FOR LINE linevar IN file MATCHING pattern DO scriptExecute the command for all lines of specified text file matching specified pattern

Arguments

ArgumentValue
foldervarthe variable into the folder item (its path relative to the workspace) name shall be written
filevarthe variable into the file item (its path relative to the workspace) shall be written
linevarthe variable into the line item shall be written
filethe file from which lines shall be read
folderthe folder in which files/subfolders shall be listed
patterna regex expression by which the found items are filtered, the wildcard character # is resolved to [0-9]+ i.e. matches numbers
scriptthe script to be executed for each valid item

Plugin

This command is defined in BasePlugin
