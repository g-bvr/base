DokumentationCommand

Purpose

Create a set of html file for documentation.

Variants

UsageDescription
DOCUMENT plugin INTO folderCreate documentation for specified plugin
DOCUMENT RELEASE release INTO folderCreate documentation for all enabled plugins

Arguments

ArgumentValue
pluginname of the plugin to be documented
releasename of the release to be documented
folderpath of folder (relative to current workspace) into which the documentation will be stored, the folder (and its ancestors) will be created if not existent

Plugin

This command is defined in BasePlugin
