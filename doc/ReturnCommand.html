ReturnCommand

Purpose

Return a value from an executed script back to the calling script. The returned value is accesible in the calling script via the variable RETURN_VALUE

Variants

UsageDescription
RETURN *Return a value from an executed script back to the calling script. The returned value is accesible in the calling script via the variable RETURN_VALUE

Arguments

ArgumentValue
*String to returned
