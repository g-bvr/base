GitPullOrCloneCommand

Purpose

Pull a changes of a git repository in the current workspace, if it was not cloned yet, clone it from specified remote location

Variants

UsageDescription
GIT PULL OR CLONE baseurl repositorypull or clone from default branch
GIT PULL OR CLONE baseurl repository tagpull already checked out branch or clone and check out specified branch or tag

Arguments

ArgumentValue
baseurlThe url prefix of the repository (not including the actual repo name)
repositoryThe name of the repository (which together with the base url constitutes the url of the repository)
tagOptional argument to specify either a branch or a tag which shall be checked out (if omitted the default branch will be used)

Plugin

This command is defined in BasePlugin
